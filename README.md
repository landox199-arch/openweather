

# OpenWeather

A simple weather web application that retrieves and displays real‑time current weather data for any city using the **OpenWeather API**. The app includes theme (dark/light) and unit (°C/°F) switching.

---

## Features
- Search weather by city name  
- Displays current temperature, weather conditions, humidity, and more  
- Switch between **Celsius** and **Fahrenheit**  
- Toggle between **Dark** and **Light** themes  
- Responsive and user‑friendly interface

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, etc.)  
- Internet connection to access the OpenWeather API

### Installation
```bash
git clone https://github.com/landox199-arch/openweather.git

### Installing
1. Open the project folder.  
2. Open `index.html` in your browser.

### Usage
1. Enter a **city name** in the search bar.  
2. Press **Enter** or click the **Search** button.  
3. View current weather information.  
4. Use the **unit switch** button to toggle between °C and °F.  
5. Use the **theme switch** to toggle between dark and light modes.

### How It Works
The app fetches weather data using the **OpenWeather API**, which provides current weather, forecasts, and atmospheric conditions for locations around the globe.  
To use it fully, register for an API key at [OpenWeatherMap](https://openweathermap.org/) and include it in the app configuration.
